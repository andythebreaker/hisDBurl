hisDBurl VSTO Excel Add-in

目的: 可以把 "從台灣歷史資料庫 (國史館、議會)" 下載的 CSV 加上 URL (to the image resource)

How to Install:

  1. Download (from GitHub release)
  2. Unzip
  3. Double click publicCert.cer，一路無腦確認，選擇安裝到受信任的根憑證 (不要選自動)
  4. Run setup.exe

Usage:

  1. Open CSV file (建議不要雙擊 CSV，會造成文件編號起始的 0s 在 Excel 中消失 (成為數字) 建議用開啟舊檔 (select all columns type as text))
  2. 去到增益集的工具列
  3. 依據資料庫名稱按下 gen url
  4. 去 "庶務" 按下 gen img (如果資料量很大，要等網路跑)

解釋:

gen img 會生成 image URL to the document (image) but due to some DB need login (or due to data void reason), the 3rd step in usage provides a link W/O login
